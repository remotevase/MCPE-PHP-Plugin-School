Volume 3: Config

Lesson 1: Most Basic Method

A Config is useful for many reasons, mainly for things like:
- Configuration
- Saving, Storing, Getting Data
In the "onEnable" Function we are gonna add some new lines:

@mkdir($this->getDataFolder()); $this->config = new Config ($this->getDataFolder() . "config.yml" , Config::YAML, array());

and Don't Forget: A NEW USE STATEMENT!

// Config utility in oreder to make Configs! use pocketmine\utils\Config;

Explanation:
mkdir - makes a new directory within the given parameter (or current directory) $this->config - This will be (an example) property defined for the whole PHP script excluding the 'onDisable' Function LOCAL VARIABLES like $abc cannot be Used in OTHER Functions! The mkdir code above makes a folder in your plugins folder with your config! To add(set) or remove something you do:

$this->config->set("string1"); $this->config->remove("string1");

To get something from the config for like example, a join MSG:

$this->config->get("JOIN_MSG");

NOTE: If you did for example:

$this->settings = // INSTEAD OF $this->config =

Thats ok! However, instead of $this->config->remove(), etc... it would be $this->settings->remove() since u wanted your property to be $this->setting
