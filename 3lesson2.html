Lesson 2: Better Config Method

You have learned method 1, lets see method 2: Method 2 loads a CUSTOM Config THAT YOU made! So, that's less coding for you since you dont have to waste time puting strings within the array on the onEnable function: Here it is

// Makes Your Config Directory @mkdir($this->getDataFolder()); // Saves CUSTOM Config $this->saveResource("config.yml"); // MAKES Custom Config $this->config = new Config($this->getDataFolder(). "config.yml", Config::YAML);
