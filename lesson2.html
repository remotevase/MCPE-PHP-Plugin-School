Lesson 2: Enable and Disable and Load Functions!

After the class statement press ENTER key and type

public function onLoad(){

If you want a Load Message! Press ENTER after the function line and type:

$this->getLogger()->info("Plugin Loading");

You just created a public function and created a load message. Here is the layout so far including all public function im going to mention in this lesson:

getLogger()->info("Plugin Loading"); } public function onEnable(){ $this->getLogger()->info("Enabled Plugin"); } public function onDisable(){ $this->getLogger()->info("Plugin Disabled"); } }

Public function onEnable is when your plugin Enables on the Logger or as we all know the CONSOLE. The plugin sends a message to the console on Enable! Public function on Disable is when your plugin disables on the Logger or as we also know the CONSOLE. The plugin sends a message to the console on Disable!
